Box Types in HTML5

Box-Level

Starts on a new row and occupies all the device-width

  • <div>
  • <h1> ... <h6>
  • <p>
  • <main>
  • <header>
  • <nav>
  • <article>
  • <aside>
  • <footer>
  • <form>
  • <video>
